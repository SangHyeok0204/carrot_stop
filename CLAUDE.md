  👥 역할별 사용자 흐름

  1. 광고주 (Advertiser)

  회원가입 → 대시보드 → 캠페인 생성 → 지원자 검토 → 인플루언서 선택 → 콘텐츠 리뷰
  ┌──────┬───────────────────────────┬──────────────────────────┐
  │ 단계 │          페이지           │           설명           │
  ├──────┼───────────────────────────┼──────────────────────────┤
  │ 1    │ /auth/signup              │ 회원가입 (이메일/Google) │
  ├──────┼───────────────────────────┼──────────────────────────┤
  │ 2    │ /advertiser/dashboard     │ 프로필 + 캠페인 통계     │
  ├──────┼───────────────────────────┼──────────────────────────┤
  │ 3    │ /advertiser/campaigns/new │ 3단계 캠페인 생성 폼     │
  ├──────┼───────────────────────────┼──────────────────────────┤
  │ 4    │ /campaigns/[id]           │ 지원자 목록 확인         │
  ├──────┼───────────────────────────┼──────────────────────────┤
  │ 5    │ 선택 API                  │ 인플루언서 선정          │
  ├──────┼───────────────────────────┼──────────────────────────┤
  │ 6    │ 제출물 리뷰               │ 승인/수정요청            │
  └──────┴───────────────────────────┴──────────────────────────┘
  2. 인플루언서 (Influencer)

  회원가입 → 피드 탐색 → 캠페인 지원 → 선정 대기 → 콘텐츠 제출 → 피드백 확인
  ┌──────┬────────────────────────────┬────────────────────────────────────┐
  │ 단계 │           페이지           │                설명                │
  ├──────┼────────────────────────────┼────────────────────────────────────┤
  │ 1    │ /auth/signup               │ 회원가입 (팔로워 수, 플랫폼 입력)  │
  ├──────┼────────────────────────────┼────────────────────────────────────┤
  │ 2    │ /influencer/feed           │ 캠페인 탐색 (필터: 목적/채널/예산) │
  ├──────┼────────────────────────────┼────────────────────────────────────┤
  │ 3    │ /influencer/campaigns/[id] │ 캠페인 상세 + 지원                 │
  ├──────┼────────────────────────────┼────────────────────────────────────┤
  │ 4    │ /influencer/applications   │ 지원 현황 (칸반 5컬럼)             │
  ├──────┼────────────────────────────┼────────────────────────────────────┤
  │ 5    │ 제출 API                   │ 포스트 URL, 스크린샷, 지표 제출    │
  ├──────┼────────────────────────────┼────────────────────────────────────┤
  │ 6    │ 결과 확인                  │ APPROVED / NEEDS_FIX               │
  └──────┴────────────────────────────┴────────────────────────────────────┘
  3. 관리자 (Admin)

  로그인 → 대시보드 → 캠페인 승인/거절 → 계약 모니터링
  ┌──────┬──────────────────┬──────────────────────────────┐
  │ 단계 │      페이지      │             설명             │
  ├──────┼──────────────────┼──────────────────────────────┤
  │ 1    │ /admin/dashboard │ 전체 통계 (캠페인/계약 현황) │
  ├──────┼──────────────────┼──────────────────────────────┤
  │ 2    │ /admin/campaigns │ 캠페인 승인/거절             │
  ├──────┼──────────────────┼──────────────────────────────┤
  │ 3    │ /admin/contracts │ 지연 계약 추적               │
  └──────┴──────────────────┴──────────────────────────────┘

  🔄 캠페인 생명주기

  DRAFT → GENERATED → REVIEWED → CLARIFYING → APPROVED → OPEN → MATCHING → RUNNING → COMPLETED
                                                                                     ↘ FAILED/CANCELLED

  현재 구현 기준 주요 흐름:

  1. 생성: 광고주가 3단계 폼 작성 → OPEN 상태로 바로 공개
  2. 지원: 인플루언서가 지원 → APPLIED 상태
  3. 선정: 광고주가 인플루언서 선택 → SELECTED
  4. 제출: 인플루언서가 콘텐츠 제출 → SUBMITTED
  5. 리뷰: 광고주가 승인 또는 수정요청 → APPROVED / NEEDS_FIX
  6. 완료: 마감일 도래 시 자동 전환 → COMPLETED
