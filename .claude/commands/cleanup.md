# cleanup

코드 정리(cleanup) 전용 커맨드. 로직 변경 없이 불필요한 코드만 제거한다.

## 수행 작업

1. **디버깅 코드 제거**
   - `console.log`, `console.warn`, `console.error` (의도적 에러 핸들링 제외)
   - `print`, `debugger`
   - `// TODO`, `// FIXME`, `// HACK` 주석

2. **미사용 import 제거**
   - 파일 내에서 실제로 참조되지 않는 import 문 삭제
   - 타입 import 중 사용되지 않는 것 포함

3. **죽은 코드 제거**
   - 주석 처리된 코드 블록 (의미 있는 설명 주석은 유지)
   - 도달 불가능한 코드
   - 사용되지 않는 변수/함수 선언

## 금지 사항

- 로직 수정, 리팩토링, 구조 변경 절대 금지
- 새로운 기능이나 개선 사항 추가 금지
- 동작하는 코드의 스타일 변경 금지
- 의미 있는 주석(JSDoc, 설명 주석) 삭제 금지

## 출력 형식   

```
[수정된 파일 코드]

---
### 변경 사항
- 제거: console.log 3개 (라인 12, 45, 78)
- 제거: 미사용 import 'useState' (라인 2)
- 제거: 주석 처리된 코드 블록 (라인 23-31)
```
